# datepicker-with-react

use

babel after `7.4.0` => `7.12.9`

webpack `5.6.0`

react react-dom `17.0.1`

and use `react-refresh`

---

```
yarn install

# run dev mode in localhost
npm start
```

homepage will in http://localhost:8080/
